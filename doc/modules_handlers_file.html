Handler: Static Content

The file handler serves files from its document root. No more, no less.

Parameters

Parameters Type Description

iocache

Boolean

Optional. Default: Enabled.

By default it will use an internal I/O cache to improve the server performance. If you donâ€™t want to use it with a directory just unset it using this parameter.

It is a good idea to disable to I/O cache if the content of the directory changes often.
